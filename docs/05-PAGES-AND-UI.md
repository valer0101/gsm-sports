# GSM Sports Platform — Pages & UI Structure

## 1. Карта сайта (Sitemap)

├── /auth
│   ├── /login                       Логин

│   ├── /sports                      Все виды спорта
│   └── /sports/[slug]               Страница вида спорта (armwrestling)

│       ├── /athletes                Атлеты вида спорта

│   └── /athletes/[id]              Профиль атлета

├── /news
│   ├── /news                        Все новости

│
├── /videos
│   ├── /videos                      Все видео

│

│   ├── /admin/dashboard             Обзор (статистика)
│   ├── /admin/tournaments           Управление турнирами

## 8. Дизайн система

### Цветовая палитра
```
Primary:      #C8102E (красный — GSM brand)
Secondary:    #1A1A2E (тёмно-синий)
Accent:       #FFD700 (золотой — для чемпионов/рейтингов)
Background:   #0F0F1A (тёмный фон)
Surface:      #1A1A2E (карточки)
Text Primary: #FFFFFF
Text Secondary: #A0A0B0
Success:      #22C55E
Error:        #EF4444
Warning:      #F59E0B
```

### Типографика
```
Заголовки:    Inter (700, 600)
Текст:        Inter (400, 500)
Цифры/счёт:  JetBrains Mono (для таблиц, сеток)
```

### Принципы UI
- Dark theme по умолчанию (спортивная тематика)
- Карточный дизайн для новостей, атлетов, турниров
- Анимации при переходах и обновлении данных
- Skeleton loading для всех асинхронных данных
- Responsive: mobile-first, breakpoints: 640px / 768px / 1024px / 1280px
